# KeyBasedSteganography

KeyBasedSteganography is a web-based application that hides secret text inside an image using image steganography techniques. The system allows users to embed a hidden message into an image using a specific key and later retrieve the message by providing the same key.

This project demonstrates how digital images can be used as a medium for secure data hiding. It provides a simple interface where users can upload an image, hide confidential information, and extract it when needed using the correct key.

## Features
- Hide secret text inside an image
- Retrieve hidden text using the correct key
- Simple web interface for encoding and decoding
- Demonstrates basic steganography concepts
